# Lista de Compras

Este proyecto es una aplicación sencilla para gestionar una lista de compras usando JavaScript.

## Descripción

La aplicación permite:

- Agregar productos a una lista.
- Eliminar productos de la lista.
- Mostrar todos los productos agregados.
- Evitar productos duplicados.

## Conceptos aplicados

- Arreglos
- Funciones flecha
- `const` y `let`
- Métodos de arreglos como `push`, `splice`, `indexOf`, `includes` y `forEach`
- Módulos con `import` y `export`
- Template literals

## Archivos del proyecto

- `app.js`: Archivo principal que ejecuta la aplicación.
- `compras.js`: Contiene la lógica para agregar, eliminar y mostrar productos.
- `package.json`: Configuración del proyecto para usar módulos ES6.

## Cómo ejecutar el proyecto

1. Clonar o descargar el repositorio.
2. Abrir una terminal en la carpeta del proyecto.
3. Ejecutar:
