# Planificador de Viajes

Este proyecto es una práctica de introducción a ECMAScript 6.

## Descripción

La aplicación permite registrar destinos de viaje, fechas, transporte, número de personas y calcular el costo total estimado del viaje.

## Conceptos aplicados

- Uso de `const` y `let`
- Funciones flecha
- Objetos
- Arreglos
- Módulos con `import` y `export`
- Template literals
- Método `forEach`
- Parámetros por defecto

## Archivos del proyecto

- `app.js`: Archivo principal que ejecuta la aplicación.
- `viajes.js`: Contiene la lógica para registrar viajes, calcular costos y mostrar el itinerario.
- `package.json`: Configuración del proyecto para usar módulos ES6.

## Cómo ejecutar el proyecto

1. Clonar o descargar el repositorio.
2. Abrir una terminal en la carpeta del proyecto.
3. Ejecutar:
